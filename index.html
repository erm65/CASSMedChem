CASSMedChem links

Sequence

Uniprot: https://www.uniprot.org/

  • The mission of UniProt is to provide the scientific community with a comprehensive, high-quality and freely accessible resource of protein sequence and functional information
  • I use it to find protein sequences

Blast: https://blast.ncbi.nlm.nih.gov/Blast.cgi

  • BLAST finds regions of similarity between biological sequences. The program compares nucleotide or protein sequences to sequence databases and calculates the statistical significance.
  • Protein BLAST (https://blast.ncbi.nlm.nih.gov/Blast.cgi?PROGRAM=blastp&PAGE_TYPE=BlastSearch&LINK_LOC=blasthome) allows the search of similar sequence in PDB database

Protein Data Bank in Europe

  • Sequence similarity search: https://www.ebi.ac.uk/pdbe-srv/PDBeXplore/sequence/
    • This PDB browser (PDBeXplore) is an interface that enables you to retrieve and analyse information on all the protein structures available in the Protein Data Bank (PDB) whose sequence display similarities to a sequence that you provide

Smart: http://smart.embl.de/

  • SMART (a Simple Modular Architecture Research Tool) allows the identification and annotation of genetically mobile domains and the analysis of domain architectures

Alignment

GUIDANCE 2: http://guidance.tau.ac.il/

  • Server for alignment confidence score

MAFFT: https://mafft.cbrc.jp/alignment/server/

  • Multiple alignment program for amino acid or nucleotide sequences

ProbCons: http://probcons.stanford.edu/index.html

  • PROBCONS is an efficient protein multiple sequence alignment program, which has demonstrated a statistically significant improvement in accuracy compared to several leading alignment tools
  • The server is down but it is possible to download the software and compile it under Linux with some modifications (09/07/2020)
    • size_t modified in std::size in all cc files
    • #include <string> in #include <cstring> in cc files (even if it works changing only in ProjectPairwais.cc, Main.cc and CompareToRef.cc)

PhyreRisk: http://phyrerisk.bc.ic.ac.uk/home

  • PhyreRisk is a dynamic web application developed to enable the exploration and mapping of genetic variants onto experimental and predicted structures of proteins and protein complexes
  • PhyreRisk integrates data from several public domain and in-house databases with information about diseases, genetic variation, biological pathways and protein structure and then facilitate the informed analysis by a wide community of users throughout the world.
  • Alsin: http://phyrerisk.bc.ic.ac.uk/search?action=fresh-search&searchTerm=Alsin

EMBL

  • Pfam: https://pfam.xfam.org/
    • The Pfam database is a large collection of protein families, each represented by multiple sequence alignments and hidden Markov models (HMMs)

Comparative modeling

Modeller: https://salilab.org/modeller/

  • MODELLER is used for homology or comparative modeling of protein three-dimensional structures
  • Tutorial basic: https://salilab.org/modeller/tutorial/basic.html
  • Tutorial basic: http://vlab.amrita.edu/?sub=3&brch=275&sim=1481&cnt=2

Oligomerization/PPI modeling

HOMCOS: http://ipproo.protein.osaka-u.ac.jp/homcos/cgi-bin/index.cgi?LANG=en

  • HOMCOS(HOMology modeling of COmplex Structure) is a server for modeling complex 3D structures using 3D molecular similarities based on template complex 3D structures in PDB

GalaxyWeb: http://galaxy.seoklab.org/

  • A web server for protein structure prediction, refinement, and related methods
  • Tool used in Alsin modelisation http://galaxy.seoklab.org/cgi-bin/submit.cgi?type=DOCK

SPRING: https://zhanglab.ccmb.med.umich.edu/spring/

  • SPRING is a template-base algorithm for protein-protein complex structure prediction.

ProtCID: http://dunbrack2.fccc.edu/ProtCiD/default.aspx

  • ProtCID contains comprehensive, PDB-wide structural information on the interactions of proteins and individual protein domains with other molecules, including four types of interactions: chain interfaces, Pfam domain interfaces, Pfam-peptide interfaces and Pfam-ligand/nucleic acids interactions

ZDock server: http://zdock.umassmed.edu/

  • ZDOCK and M-ZDOCK are Fast Fourier Transform based protein docking programs
  • Tool used in Alsin modelisation
